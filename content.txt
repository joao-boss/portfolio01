The Build Plan (How it works under the hood)
The Parser: The Agent creates a script that scans content.txt for headers like [TECHNICAL_STACK]. It recognizes that Ableton_Live: 98 means the label is "Ableton Live" and the variable is 98.

The Component Mapping:

Text Data → Sent to the Hero section.

Placeholders → Sent to the Grid sections (awaiting your real links).

Numerical Data → Sent to the Fader Component to define the translate-y or bottom CSS property of the fader knob.

The Visual Engine: The Agent uses Tailwind to create the "Studio" look—adding shadows to the fader tracks so they look recessed into the "desk," and adding a metallic gradient to the knobs.
